Sobre o projeto


Este projeto faz parte da Projeto ONE desenvolvido pela Oracle Next Education e Alura.


Esse é o primeiro Challenge Oracle ONE - Iniciante em Programação que tem como objetivo o desenvolvimento de um decodificador de textos com a utilização das tecnologias HTML, CSS e Javacript além de nos fornecer conhecimentos sobre a utilização do Trello como ferramenta de acompanhamento do desenvolvimento e do Git e Github para versionamento de código.
